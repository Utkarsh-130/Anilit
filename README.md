# Anilit 

A sleek, high-octane anime and manga tracking application inspired by the iconic **Persona 5** aesthetic. Built with **React Native** and **NativeWind**, Anilit turns the simple task of browsing media into a stylish "Metaverse" experience.

## ✨ Features

*   **Persona 5 Aesthetic:** High-contrast layouts with jagged-edged cards and bold typography that capture the "Take Your Heart" energy.
*   **Comprehensive Discovery:** Browse a vast library of anime and manga using the **Jikan API**.
*   **Dynamic Tag Filtering:** Filter through genres like Action, Mystery, and Sci-Fi with a stylized, easy-to-use interface.
*   **Real-time Metadata:** Stay updated with current ratings, episode/chapter counts, and detailed synopses.
*   **Native Performance:** Utility-first styling with **NativeWind** ensures smooth navigation and transitions.

## 🛠 Tech Stack

*   **Framework:** React Native
*   **Styling:** NativeWind (Tailwind CSS for React Native)
*   **API:** Jikan API (Unofficial MyAnimeList API)
*   **Environment:** Expo

## 🚀 Getting Started

### Prerequisites
*   Node.js
*   Expo Go app on your mobile device or an Android/iOS emulator

### Installation
1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Utkarsh-130/anilit.git
    cd anilis
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Start the application:**
    ```bash
    npx expo start
    ```
